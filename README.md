# Choperia Jacques — Cardápio Digital

App do cardápio (Next.js + Supabase). Publicação 100% online via Vercel.

## Variáveis de ambiente (configurar na Vercel)
- `NEXT_PUBLIC_SUPABASE_URL` — Project URL do Supabase
- `NEXT_PUBLIC_SUPABASE_ANON_KEY` — chave anon public do Supabase

## Publicar
1. Suba estes arquivos num repositório no GitHub.
2. Na Vercel: New Project > Import do repositório.
3. Em Environment Variables, adicione as duas chaves acima.
4. Deploy. Pronto.

Admin: toque longo em "Desde 2013" no rodapé e faça login com o e-mail/senha
criados em Authentication > Users no Supabase.
